Kali 个性化配置

发表于2023-06-25|Linux
|字数总计:2.1k|阅读时长:9分钟|阅读量:

  • 由于个人强迫症的原因，Kali原版用不太习惯，这里就进行简单的美化和应用安装。

  • 版本来源：官网上新版本的 vmware 解压使用版，这里以 kali-linux-2023.2-vmware-amd64 为例。

简单美化

添加网卡

  • 点击编辑虚拟机配置，添加2张网卡：
    • 网络适配器：桥接
    • 网络适配器2：NAT
    • 网络适配器3：VMnet 1
  • 如果用不到 NAT 和 VMnet 1，临时禁用这两张网卡：

中文调整

  • 安装中文字体、设置系统语言区
  • 之后会弹出选择界面，下滑到底，空格选中 zh_CN.UTF-8 UTF-8 回车后，再次选择 zh_CN.UTF-8 继续回车即可。
  • 安装小企鹅输入法，安装完成后，重启即可：
  • 每次需要使用时按 Ctrl+space 可以切换输入法。
  • 后续会遇到“将标准文件夹更新到当前语言吗？”，勾选“不要再次询问我”，点击“保留旧的名称”。

桌面美化

桌面壁纸

  • 右键桌面设置 -> 目录 -> 选择你的图片目录即可。

图标整理

  • 右键桌面设置 -> 图标 -> 取消默认图标。

任务栏调整

  • 右键任务栏 -> 面板 -> 面板首选项 -> 锁定面板 -> 拖拽任务栏至最下方 -> 锁定面板 -> 对于不需要的图标进行 remove 即可 -> 时间修改为 24 小时制（时区）

外观改变

  • 外观 -> 样式 -> Adwaite-dark

文件夹显示

  • 打开文件夹 -> 视图 -> 列表视图 -> 按所需放大

电源管理

  • 右键应用程序 -> 设置 -> 电源管理器 -> 显示 -> 将使用电池和插入电源三个选项全部调成从不即可。

锁屏修改

  • 设置 -> LightDM 桌面管理器 -> 背景 -> 进入/usr/share/backgrounds/kali -> 选取一个（自定义图片会报错，我选 cubism）
